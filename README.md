Js Boilerplate :

- Webpack
- React
- babel / es2015
- karma / mocha / jsdom
- flow
- eslint / nsp


### Dev mode

    > npm start

### Build mode

    > npm run build

### Test mode

unit test:

    > npm test

eslint:

    > npm test:lint

typecheck:

    > npm test:typecheck

vulnerabilities scan for node_modules:

    > npm test:audit
